CSS 单位说明

px (像素)

绝对单位，1px = 1/96 英寸。最常用的单位，但不利于响应式设计。

rem (根 em)

相对于根元素 (html) 的 font-size。推荐用于响应式布局。

em

相对于父元素的 font-size。会层层继承，使用时需谨慎。

vw / vh

视口宽度/高度的 1%。适合全屏布局和响应式字体。

vmin / vmax

视口宽高中较小/较大值的 1%。适合保持元素比例。

% (百分比)

相对于父元素的对应属性。width: 50% 表示父元素宽度的一半。
