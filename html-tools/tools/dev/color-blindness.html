色盲类型说明

红色盲 (Protanopia)

无法感知红色光，约占男性 1%。红色看起来像暗黄色或棕色。

绿色盲 (Deuteranopia)

无法感知绿色光，是最常见的色盲类型，约占男性 6%。

蓝色盲 (Tritanopia)

无法感知蓝色光，非常罕见，约占人群 0.01%。

全色盲 (Achromatopsia)

完全无法感知颜色，只能看到灰度，极为罕见。
