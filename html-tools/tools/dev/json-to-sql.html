使用说明

支持的输入格式

  • 单个对象： {"name": "张三", "age": 25}
  • 对象数组： [{"name": "张三"}, {"name": "李四"}]

数据库差异

  • MySQL ：支持 INSERT IGNORE, REPLACE INTO, 批量插入
  • PostgreSQL ：使用 ON CONFLICT DO NOTHING
  • SQLite ：支持 INSERT OR IGNORE, INSERT OR REPLACE
  • SQL Server ：使用方括号包裹标识符
  • Oracle ：使用 INSERT ALL 语法

注意事项

  • 字符串值会自动转义单引号
  • null 值会生成 NULL
  • 布尔值根据数据库类型转换
  • 嵌套对象会转为 JSON 字符串
