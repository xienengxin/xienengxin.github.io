🖥️ MCP 客户端大全

支持 Model Context Protocol 的 AI 客户端和开发工具

💡 什么是 MCP 客户端？

MCP 客户端是支持 Model Context Protocol 的应用程序，可以连接 MCP 服务器来扩展 AI 的能力。通过 MCP，AI 可以访问文件系统、数据库、API 等外部资源。

相关页面： MCP 配置指南 · 官方文档： modelcontextprotocol.io

Claude Desktop

Anthropic 官方
Anthropic 官方桌面客户端，原生支持 MCP，是最完整的 MCP 实现。
  • 原生 MCP 支持，功能最完整
  • 支持 Claude 3.5/3 全系列模型
  • 支持 Artifacts 和项目功能
  • 支持图片、PDF 文件上传
