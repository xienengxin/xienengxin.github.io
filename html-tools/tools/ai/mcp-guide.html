🔌 MCP 配置指南

Model Context Protocol 配置教程与热门服务器

💡 什么是 MCP？

MCP (Model Context Protocol) 是 Anthropic 开发的开源标准协议，用于连接 AI 应用与外部系统。通过 MCP，Claude 可以访问文件系统、数据库、API 等外部资源，大幅扩展其能力边界。

官方文档： modelcontextprotocol.io · GitHub： github.com/modelcontextprotocol

🖥️ Claude Desktop 配置

配置文件位置：

macOS 
~/Library/Application Support/Claude/claude_desktop_config.json
Windows 
%APPDATA%\Claude\claude_desktop_config.json

配置示例：

claude_desktop_config.json 
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path/to/allowed/dir"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "your_token_here"
      }
    }
  }
}
💡 提示： 修改配置后需要重启 Claude Desktop 才能生效。

⌨️ Claude Code 配置

Claude Code 使用 .mcp.json 文件配置 MCP 服务器，可以放在项目根目录或用户目录。

项目级配置：.mcp.json 
{
  "mcpServers": {
    "memory": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-memory"]
    },
    "fetch": {
      "command": "uvx",
      "args": ["mcp-server-fetch"]
    }
  }
}

也可以通过命令行添加：

命令行 
# 添加 MCP 服务器
claude mcp add memory -- npx -y @modelcontextprotocol/server-memory

# 查看已配置的服务器
claude mcp list

# 移除服务器
claude mcp remove memory

🌟 热门 MCP 服务器

官方和社区提供的常用 MCP 服务器：

📁
Filesystem
安全的文件系统访问，支持读写文件、创建目录、搜索文件等操作。
安装命令
npx -y @modelcontextprotocol/server-filesystem /path/to/dir 
"filesystem": {
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/you/projects"]
}
官方 文件操作
GitHub
🐙
GitHub
GitHub API 集成，支持仓库管理、Issue、PR、代码搜索等功能。
安装命令
npx -y @modelcontextprotocol/server-github
"github": {
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-github"],
  "env": {
    "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_xxxx"
  }
}
⚠️ 需要设置 GITHUB_PERSONAL_ACCESS_TOKEN
官方 API
GitHub
🔀
Git
Git 仓库操作，支持克隆、提交、分支管理、查看历史等。
安装命令
uvx mcp-server-git --repository /path/to/repo
"git": {
  "command": "uvx",
  "args": ["mcp-server-git", "--repository", "/path/to/repo"]
}
官方 版本控制
GitHub
🌐
Fetch
网页内容抓取，支持获取网页内容并转换为 Markdown 格式。
安装命令

"fetch": {
  "command": "uvx",
  "args": ["mcp-server-fetch"]
}
官方 网络
GitHub

Memory
知识图谱记忆系统，让 Claude 能够持久化存储和检索信息。
安装命令

"memory": {
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-memory"]

官方 存储
GitHub
🕐
Time
时间和时区工具，获取当前时间、时区转换等功能。
安装命令
uvx mcp-server-time
"time": {
  "command": "uvx",
  "args": ["mcp-server-time", "--local-timezone", "Asia/Shanghai"]
}
官方 工具
GitHub

SQLite
SQLite 数据库操作，支持查询、分析本地数据库文件。
安装命令
uvx mcp-server-sqlite --db-path /path/to/db.sqlite
"sqlite": {
  "command": "uvx",
  "args": ["mcp-server-sqlite", "--db-path", "/path/to/database.db"]
}
官方 数据库
GitHub
🐘
PostgreSQL
PostgreSQL 数据库连接，支持查询和数据分析。
安装命令
npx -y @modelcontextprotocol/server-postgres postgres://... 
"postgres": {
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-postgres", "postgres://user:pass@localhost/db"]
}
⚠️ 连接字符串格式: postgres://user:password@host:port/database

GitHub
