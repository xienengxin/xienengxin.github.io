✨ Claude Skills 精选

Anthropic 官方 Skills 集合，提升 Claude 在特定任务上的表现

💡 什么是 Claude Skills？

Skills 是包含指令、脚本和资源的文件夹，Claude 会动态加载以提升特定任务的处理能力。它们可以教 Claude 按照特定方式完成任务，无论是创建符合品牌规范的文档、使用组织特定工作流分析数据，还是自动化个人任务。

官方仓库： github.com/anthropics/skills

/plugin install example-skills@anthropic-agent-skills
/plugin install document-skills@anthropic-agent-skills
