🎯 什么是 Claude Skills

Claude Skills 是 Anthropic 官方推出的技能系统，允许 Claude 动态加载一组指令、脚本和资源，以增强其在特定任务上的执行能力。简单来说，Skills 就像是给 Claude 安装的"插件"或"专业技能包"。

每个 Skill 是一个包含 SKILL.md 文件的文件夹，文件中定义了：

  • 元数据 ：技能名称、描述、触发条件
  • 指令 ：Claude 执行任务时遵循的详细 SOP
  • 资源 ：脚本、模板、参考文档等

✨ 为什么要用 Skills

💡 核心价值

Skills 让 Claude 从"通才"变成"专家"，在特定领域提供更精准、更专业的输出。

主要优势：

  • 专业化输出 ：针对特定任务（如 PDF 处理、演示文稿制作）提供专业级结果
  • 一致性 ：确保同类任务的输出风格和质量一致
  • 效率提升 ：预置的工作流和脚本减少重复沟通
  • 可复用 ：一次配置，团队共享
  • 可扩展 ：支持自定义技能，满足个性化需求

📦 安装方法

方法一：通过 Claude Code 命令行安装

这是最推荐的安装方式，适用于 Claude Code 用户：

bash 
# 1. 添加官方插件市场
/plugin marketplace add anthropics/skills

# 2. 安装文档技能包（Word、Excel、PDF、PPT）
/plugin install document-skills@anthropic-agent-skills

# 3. 安装示例技能包（创意设计、开发工具等）
/plugin install example-skills@anthropic-agent-skills

方法二：通过交互式菜单安装

步骤 
1. 在 Claude Code 中输入 /plugin
2. 选择 "Browse and install plugins"
3. 选择 "anthropic-agent-skills"
4. 选择需要的技能包（document-skills 或 example-skills）
5. 选择 "Install now"

方法三：Claude.ai 网页版

如果你是 Claude Pro/Team 订阅用户，可以直接在 claude.ai 使用所有示例技能，无需额外安装。也可以上传自定义技能。

⚠️ 注意

文档技能（docx、xlsx、pdf、pptx）为源代码可见但非开源，遵循 Anthropic 使用条款。示例技能采用 Apache 2.0 开源许可。

🚀 使用方法

安装技能后，只需在对话中 提及技能名称 即可自动触发：

示例对话 
# 使用 PDF 技能
"Use the PDF skill to extract the form fields from ./contract.pdf"

# 使用 XLSX 技能
"用 xlsx 技能帮我分析这个销售数据表格"

# 使用前端设计技能
"Use frontend-design to create a modern dashboard component"
💡 技巧

你可以直接描述任务，Claude 会自动识别并调用合适的技能。不需要每次都显式提及技能名称。

🛠️ 自定义技能

创建自定义技能非常简单，只需一个包含 SKILL.md 文件的文件夹：

markdown - SKILL.md 
---
name: my-custom-skill
description: 这是一个自定义技能的描述，说明什么时候应该使用它
---

# My Custom Skill

## 概述
[技能的详细说明]

## 使用场景
- 场景 1
- 场景 2

## 执行指南
1. 第一步操作
2. 第二步操作

## 输出格式
[定义输出的格式要求]

## 示例
[提供具体示例]

技能结构

目录/文件 必需 说明
SKILL.md 是 技能定义文件，包含 YAML frontmatter 和指令
scripts/ 否 可执行脚本（Python、Bash 等）
references/ 否 参考文档（API 文档、知识库等）
assets/ 否 静态资源（模板、图标、字体等）

必填字段

字段 说明
name 技能唯一标识符（小写，用连字符分隔，如 my-skill ）
description 技能功能和使用场景的完整描述，Claude 据此判断何时调用
