💡 sitemap.xml 说明

  • loc - 页面的完整 URL（必需）
  • lastmod - 最后修改时间，格式：YYYY-MM-DD
  • changefreq - 更新频率：always, hourly, daily, weekly, monthly, yearly, never
  • priority - 优先级：0.0 到 1.0，默认 0.5，首页通常设为 1.0
  • sitemap.xml 最多包含 50,000 个 URL 或文件大小不超过 50MB
  • 记得在 robots.txt 中添加 Sitemap 指令
