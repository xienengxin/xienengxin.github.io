公式说明

均值 (Mean) : 所有数据的算术平均值

中位数 (Median) : 将数据排序后处于中间位置的值

众数 (Mode) : 出现次数最多的值

方差 (Variance) : 数据与均值差的平方的平均值，衡量离散程度

标准差 (Standard Deviation) : 方差的平方根，更直观的离散度量

四分位数 : Q1为25%分位数，Q3为75%分位数

四分位距 (IQR) : Q3 - Q1，衡量数据的中间50%的分散程度
